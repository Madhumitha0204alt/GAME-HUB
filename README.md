Game Hub is a console-based multi-game platform built entirely in C++. 

The idea behind this
project is simple but effective — instead of writing separate programs for every game, why
not bring them all under one roof? This project does exactly that, offering three classic games:
Tic Tac Toe, Connect 4, and Snake ; within a single menu-driven application.

The project was built with a strong emphasis on Object-Oriented Programming (OOP)
principles such as inheritance, polymorphism, and encapsulation.

Each game is implemented as a class that inherits from a common base class called Game, making the codebase clean,
extensible, and easy to understand.

Beyond just gameplay, the application also features a file-based leaderboard system that
records every result — wins, draws, and timeouts  to a text file called scores.txt. Players
are tracked by username, and the leaderboard can be viewed or cleared at any time from the
main menu.

This project demonstrates how C++ concepts learned in the classroom can be applied to build
something genuinely interactive and fun.
